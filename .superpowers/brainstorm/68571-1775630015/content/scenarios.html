全场景动效应用方案

每个 UI 交互场景的具体动效方案

🔀 1. 页面切换（workspace / connection / dashboard）

动效：fadeInUp — opacity(0→1) + translateY(8px→0)
时长：var(--motion-duration-slow) = 0.3s
缓动：var(--motion-ease-out)
实现：在页面根容器添加 .motion-fade-up-enter 动画类
文件：src/pages/main/index.tsx (路由切换处)

📑 2. Tab 标签页管理

Tab 切换：
  内容区 fadeIn — opacity(0→1), duration: 0.2s, ease-out
新增 Tab：
  Tab 标签 fadeIn + 内容区 fadeInUp, duration: 0.2s
关闭 Tab：
  Tab 标签 fadeOut + width 收缩, duration: 0.15s
文件：src/pages/main/workspace/components/WorkspaceTabs/

🌲 3. Tree 数据库导航

节点展开/收起：
  collapse 动画 — max-height + opacity, duration: 0.2s, ease-in-out
  箭头旋转 — transform: rotate(0→90deg), duration: 0.2s
搜索结果出现：
  stagger fadeIn — 每项延迟 30ms, duration: 0.15s
节点 hover：
  background-color transition, duration: 0.1s
文件：src/blocks/Tree/, src/pages/main/workspace/components/WorkspaceLeft/
现状：已有基础 transition，需要统一到 motion token

🪟 4. Modal 弹窗

打开：
  遮罩层 fadeIn(0.2s) + 内容 zoomIn scale(0.85→1) + fadeIn(0.3s)
关闭：
  遮罩层 fadeOut(0.2s) + 内容 zoomOut scale(1→0.85) + fadeOut(0.2s)
实现：Ant Design Modal 已有内置动效，自定义 Modal (BaseModal/TriggeredModal) 需添加
文件：src/components/BaseModal/, src/components/TriggeredModal/

📥 5. Drawer 抽屉

日志面板（底部）：slideUp + fadeIn, duration: 0.3s, ease-out-circ
侧边 Drawer：slideRight + fadeIn, duration: 0.3s, ease-out-circ
实现：Ant Design Drawer 内置动效，确保参数与 token 一致
文件：src/blocks/LogViewer/

📊 6. 数据表 & 查询结果

查询执行 → 结果出现：fadeInUp, duration: 0.2s
分页切换：fadeIn 内容过渡, duration: 0.15s
行编辑保存：行背景 flash 高亮 (0.5s 绿色闪烁后恢复)
行删除：fadeOut + height 收缩, duration: 0.2s
Loading 状态：fadeIn/Out 切换（已有 Loading 组件，保持不变）
文件：src/components/SearchResult/, src/components/Output/

📐 7. 侧边栏 & 面板

左侧面板拖拽调整：实时跟随（已有 DraggableContainer）
面板折叠/展开：width transition 0.3s ease-in-out + 内容 fadeIn
Save List 面板切换：fadeIn 0.2s（在 WorkspaceLeft 中切换显示）
文件：src/components/DraggableContainer/, src/pages/main/workspace/components/WorkspaceLeft/

🔔 8. 通知 & 消息

Message 消息：slideDown + fadeIn 从顶部进入, 0.3s
Notification 通知：slideRight + fadeIn 从右侧进入, 0.3s
自动消失：fadeOut + slideUp, 0.2s
实现：Ant Design message/notification 已有内置动效，确认使用统一 token
文件：src/components/MyNotification/

🔗 9. 连接管理

连接列表加载：stagger fadeInUp, 每项延迟 50ms
连接状态变化：状态图标 fadeIn 切换, 0.2s
新建/编辑连接弹窗：同 Modal 动效规范
文件：src/pages/main/connection/, src/components/ConnectionEdit/

⌨️ 10. 通用交互

按钮 hover/active：background-color + box-shadow transition 0.1s
图标按钮 hover：background-color 0.15s（已有，统一 token）
输入框 focus：border-color + box-shadow 0.2s（Ant Design 内置）
右键菜单：zoomIn + fadeIn, origin: 点击位置, 0.15s
Tooltip/Popover：fadeIn + 微位移, 0.1s（Ant Design 内置）

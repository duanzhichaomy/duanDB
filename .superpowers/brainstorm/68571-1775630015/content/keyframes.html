全局 Keyframe 动画库

在 motion.less 中定义的可复用 @keyframes 动画

1. Fade 淡入淡出

fadeIn / fadeOut
fadeIn

opacity: 0 → 1
用途：Tooltip、通知、内容切换

fadeInUp / fadeOutDown
fadeInUp

opacity + translateY(8px → 0)
用途：页面进入、列表项出现

2. Slide 滑入滑出

slideInRight / slideOutRight
slideRight

translateX(100% → 0)
用途：Drawer、侧面板

slideInDown / slideOutUp
slideDown

translateY(-100% → 0)
用途：下拉菜单、底部Drawer

3. Zoom 缩放

zoomIn / zoomOut
zoomIn

scale(0.85) + opacity → 1
用途：Modal、Popover、右键菜单
